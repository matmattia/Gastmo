# Gastmo's A Software To Manage Orders

Gastmo è un modulo di [MatCMS](https://www.matriz.it/projects/matcms/ "Matriz | Projects | MatCMS") che permette di gestire gli ordini di un GAS (Gruppo d'Acquisto Solidale).
Sfruttando MatCMS è possibile anche costruire un sito per il proprio GAS.
